# ReactSneakers
 Простой магазин на React
# Функционал
  1. Рендер карточек товаров с mockapi
  2. Добавление тоавров в корзину, подсёт стоимости товара 
  3. Отправка заказа на бекенд(mockapi) и отображение его в разделе заказы
# СТЭК
- React
- React Router
- React Context
- react-slick
- SASS / CSS Modules
- Axios
